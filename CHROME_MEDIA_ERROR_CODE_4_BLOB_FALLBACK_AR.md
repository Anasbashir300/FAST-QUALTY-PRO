# إصلاح Chrome media error code 4

إذا كان رابط الصوت يفتح يدويًا ولكن Chrome يرفض تشغيله داخل `<audio>` ويظهر:

```text
Chrome media error code: 4
```

فإن `custom-dub-bridge.js` يجرب الآن طريقة ثانية تلقائيًا:

1. يحاول تشغيل رابط MP3 مباشرة.
2. إذا فشل، يجلب الملف عبر `fetch`.
3. يحوله إلى `Blob`.
4. يشغله كرابط محلي `blob:` داخل الصفحة.
5. عند الإيقاف يتم حذف `ObjectURL` من الذاكرة.

بهذا لا تحتاج لفتح الرابط يدويًا.
